1. Если файл update.json пустой, заполните его в следующем формате:

  {
    "last_commit": "hash"
  }

  Перейдите по ссылке:
  https://github.com/tsybtw/OF_HELPER/commits/main/

  Нажмите на кнопку копирования хеша (слева от кнопки <>), соответствующего вашей текущей версии, и вставьте его вместо слова "hash".

2. При необходимости переименуйте файлы обновления, например, в upd1 (или upd1.exe).

3. Для обновления:

  - на Mac os:

    1. Кликните правой кнопкой мыши по папке update → Открыть в Терминале.

    2. Введите команды по порядку:

    xattr -d com.apple.quarantine ./upd1
    sudo chmod +x upd1
    ./upd1

  - на Windows: 

    1. Дважды кликните по upd1.exe.

После обновления удалите файл upd1 (или upd1.exe).
Последняя версия всегда доступна по ссылке: https://github.com/tsybtw/OF_HELPER/archive/refs/heads/main.zip
После скачивания последней версии по ссылке делать обновление через файл обновления уже не нужно.
