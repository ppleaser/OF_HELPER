Переименуйте файлы для обновления, например в upd1 (upd1.exe)

Для обновления на mac os:

Правой кнопкой мыши по папке update -> Открыть папку в терминале

xattr -d com.apple.quarantine ./upd1
sudo chmod +x upd1
./upd1

Для обновления на windows: 

Дабл клик по upd1.exe (либо ./upd1 через powerShell)

Чтобы скинуть историю обновлений (либо обновить заново с нуля), отредактируйте файл update.json. 

Для этого перейдите по ссылке: https://github.com/ppleaser/OF_HELPER/commits/main/
Оттуда скопируйте хеш последнего обновления (кнопочка копирования правее от названия обновления). Все обновления после него будут загружены.

Вставьте в файл:

{
  "last_commit": "c78672ce935f378e9ffa79e8c071d43085475db2"
}

Где в кавычках будет нужный хеш (выше просто пример). 

После обновления удалите файлы upd1 (upd1.exe).
