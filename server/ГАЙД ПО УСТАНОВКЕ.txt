на macOS: Python, Node.js
на Windows: Python, Node.js, Java, Lightshot/ShareX, Windows Terminal

Запуск на macOS:

    1. Нажмите правой кнопкой мыши по папке server -> Новый терминал по адресу папки
    2. Выполните команду:

    ./main

Для первого запуска на Windows:

    1. Запустите файл wt.exe в Windows Terminal  
    2. Нажмите на значок ⏷
    3. Выберите Параметры.
    4. В меню слева выберите Defaults.
    5. Найдите Starting directory:
        - Уберите галочку "Use parent process directory".
        - Укажите путь к папке server.
    6. Нажмите Save.
    7. Нажмите + (новая вкладка).
    8. Введите команду: ./main

Для следующих запусков на Windows:

    Выполняйте только пункты 1 и 8 из инструкции выше.
