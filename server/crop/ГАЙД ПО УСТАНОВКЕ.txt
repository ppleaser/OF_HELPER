Для установки на mac os:

В файле update.sh заполнить путь в пустых кавычках на полный путь к файлу в папке templates\output.html
Для этого зайдите в папку, потом нажмите на свойства этого файла, после чего скопируйте путь и в конец добавьте \output.html

Правой кнопкой мыши по папке crop -> Открыть папку в терминале

sudo npm i -g browser-sync
