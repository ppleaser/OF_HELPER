1. Перейдите по ссылке:
https://github.com/tsybtw/OF_HELPER/releases

Для macOS:
Скачайте файлы ffmpeg и ffprobe

Для Windows: 
Скачайте файлы ffmpeg.exe и ffprobe.exe

2. Поместите 2 скачанных файла в папку ffmpeg

Для macOS:
Нажмите правой кнопкой мыши на папку crop -> Новый терминал по адресу папки

Для Windows: 
Перейдите в папку crop и нажмите на строку пути -> Введите cmd и нажмите Enter

3. Выполните команду:
node install.js

4. Перейдите на сайт:

https://my.telegram.org/auth

    1. Введите номер телефона, к которому привязан ваш Telegram, и подтвердите вход через код из приложения.
    2. После входа выберите раздел API Development Tools.
    3. Нажмите Create New Application.

    Заполните поля:
    App title - любое название на латинице (от 5 символов)
    Short name - аналогично
    Остальные поля можно оставить как есть.

    4. Нажмите Create application.

    5. Заполните файл .env:

    Файл может быть скрыт. Находясь в папке crop:
    На Windows: в настройках проводника включите показ скрытых файлов.
    на macOS: нажмите комбинацию Shift + Command + точка.

    API_ID=API ID (на сайте)
    API_HASH=API Hash (на сайте)
    PHONE=номер_телефона_от_аккаунта 
    FOLDER=/путь/к/папке/со/скриншотами
